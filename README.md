# Productivity

This workspace now contains a starter SwiftUI app for an iPhone productivity reminder tool.

## What it does
- Lets you enable periodic reminders for:
  - Exercise
  - Eye-strain relief
  - Journal notes
  - Planning the next task
- Shows a full-screen reminder overlay when a reminder is triggered.
- Includes a 5-minute focus block timer.
- Uses local notifications so the app can interrupt your current workflow.

## How to open it
Open the Xcode project file at:
- Productivity.xcodeproj

Then build and run it on your iPhone or iPhone simulator.

## Notes
- The app requests notification permission on launch.
- You can later expand this into a richer experience with custom intervals, sounds, and persistent journaling.
